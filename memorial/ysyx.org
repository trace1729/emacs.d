  1. 预学习申请表的整理
  2. 预学习面试内容准备
     [[https://docs.qq.com/doc/DR0poZWl3YmVTSWh5?dver=][入学答辩流程参考]]
     + 前期准备
       #+begin_example
       通识问卷完成（达到100分）放到答辩申请表、将学习记录及读后感文档权限设为公开

       设备检查：摄像头、麦克风、是否能正常共享屏幕（需要投屏讲解部分代码
（放到流片答辩）证件检查：需要学生身份证明（能证明本人身份，一般为学生卡或者学生证，录取通知书、毕业证书）
       #+end_example
     + 大概可以问什么问题
       #+begin_example
       1. 请同学介绍其软硬件基础，内容包括：上过的相关课程、做过的课设、做过的相关项目等

       2. PA1：
        优雅的退出、
        打印寄存器、
        扫描内存、
        监视点的实现、
        表达式求值采用的算法、
        表达式求值的实现方式、
        表达求值的测试方式、
        表达求值生成测试用例的除零用例的排除等。
       3. 数电实验：
        0基础不完成键盘的不予通过，
	如果项目经历数电相关比较突出可以酌情跳过。
	如果进度超过预学习答辩内容，适当按当前最新进度进行提问。
       4. 预期目标
         询问答辩人的预期目标（具体想做个什么样的处理器）
       #+end_example
     + 在哪里记录
       [[https://docs.qq.com/sheet/DSXhIZkVTTnJueXpm?tab=0e7mh0][答辩记录表]]
     + 如何判断时候能过
     + 有没有要反问的
     + 通过的话需要念一个承诺书
